# Product Store

A modern, full-stack web application designed for comprehensive product inventory management. Built with the MERN stack, it offers a seamless and responsive interface for users to effortlessly manage their product catalogs.

## 🎯 Target Users
Business owners, store managers, administrators, and developers looking for a lightweight, scalable solution to track, organize, and manage their digital or physical product inventory.

## 💡 Problem It Solves
Managing product catalogs manually can be tedious and prone to error. Product Store eliminates this complexity by providing a centralized, reactive application to effortlessly track products, keeping inventory up-to-date with complete CRUD (Create, Read, Update, Delete) capabilities.

---

## 🛠 Tech Stack

**Frontend:**
- React 19 (built with Vite)
- Chakra UI & Emotion (for modern, accessible styling)
- Framer Motion (for smooth micro-animations)
- Zustand (for predictable state management)
- React Router DOM (for client-side routing)

**Backend:**
- Node.js
- Express.js
- Mongoose (Object Data Modeling)

**Database:**
- MongoDB (Atlas or local instance)

---

## ✨ Features
- **Complete CRUD Functionality:** Easily create, view, edit, and delete products.
- **Robust State Management:** Powered by Zustand for optimal performance with minimal boilerplate.
- **Modern UI/UX:** Responsive application layout utilizing Chakra UI components.
- **Dynamic Theming:** Built-in Light and Dark mode options.
- **Micro-Animations:** Engaging layout transitions driven by Framer Motion.
- **Production-Ready SPA Serving:** Backend automatically routes and serves the frontend React build in production environments.

---

## 📋 Requirements
- **Node.js**: v18.0.0 or higher
- **MongoDB**: A running MongoDB instance (Local or Atlas snippet)
- **NPM Package Manager**
